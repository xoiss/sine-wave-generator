uq016_t qmul_uq016 ( const uq016_t  a,
const uq016_t  b 
)

Returns the product of two fixed point values, unsigned fixed point 0.16-bit version.

Parameters
[in]a– the first multiplicand.
[in]b– the second multiplicand.
Returns
The product a*b.

The domain of the defined function is the superposition of all allowed values of a and b, which are in turn both belong to the set of UQ0.16 values in the discrete range [0.0; 1.0-1/2^16] with resolution of 1/2^16.

The codomain of the defined function is the set of UQ0.16 values in the discrete range [0.0; 1.0-1/2^16] with resolution of 1/2^16.

The product is rounded down to the nearest value with resolution of 1/2^16.

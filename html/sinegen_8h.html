Detailed Description

Interface to the sine wave generator.

This file provides declarations for the set of functions used to manipulate the sine wave generator, and declaration of the generator descriptor data structure.

§ gen_init()

void gen_init ( struct gen_descr_t *const  pgen)

Initializes a sine wave generator.

Parameters
[in,out]pgen– pointer to the initialized generator descriptor object.

During initialization the following values assigned to the generator attributes:

  • frequency – is set to 0, which means that the generation is paused.
  • phase – is set to 0, just the initial phase.
  • attenuation – is set to 0, which means no attenuation.

§ gen_output()

sq015_t gen_output ( const struct gen_descr_t *const  pgen)

Returns the generator momentary output.

Parameters
[in]pgen– pointer to a generator descriptor object.
Returns
Momentary amplitude of the generated signal.

The output signal produced by the generator is described with the formula:

  • u(t) = sin(2*pi*Fo/Fs*t + phi0) * (1-att), where:
  • Fo – the oscillator frequency, hertz.
  • Fs – the sampling frequency, hertz.
  • phi0 – the oscillator initial phase, radians.
  • att – the momentary attenuation.
  • t – the current moment of time, in terms of sampling frequency periods.

This function shall be called each time the applications needs to evaluate the current level of the generator output signal. Generally this function is called each sampling period before the generator state is propagated with the gen_step function, or the generator attributes are reassigned with functions gen_set_xxx.

The codomain of the generator output signal levels is the set of SQ0.15 values in the discrete range [-1.0; +1.0-1/2^15] with resolution of 1/2^15. This range corresponds to the floating point range [-1; +1) with resolution of 1/2^15:

amplitude fixed point value container code
+1 not reachable (*) not reachable
+1 - 1/2^15 +0.999969482421875 0x7FFF
+0.75 +0.75 0x6000
+0.5 +0.5 0x4000
+1/2^15 +0.000030517578125 0x0001
0 0.0 0x0000
-1/2^15 -0.000030517578125 0x7FFF
-0.5 -0.5 0xC000
-1 + 1/2^15 -0.999969482421875 0x8001
-1 -1.0 0x8000
Note
The momentary amplitude value +1 exactly, which cannot be represented as a SQ0.15 value, is substituted with 1-1/2^15.

const uq016_t  att 
)

Assigns the generator output attenuation.

Parameters
[in,out]pgen– pointer to a generator descriptor object.
[in]att– a new value of the output signal momentary attenuation.

The allowed values of att are the set of UQ0.16 values in the discrete range [0.0; 1.0-1/2^16] with resolution of 1/2^16. This range corresponds to the floating point range [0; 1) with resolution of 1/2^16:

attenuation fixed point value container code
1 not allowed (*) not allowed
1 - 1/2^16 0.9999847412109375 0xFFFF
0.75 0.75 0xC000
0.5 0.5 0x8000
1/2^16 0.0000152587890625 0x0001
0 0.0 0x0000
Note
The attenuation factor value 1 exactly, which cannot be represented as a UQ0.16 value, shall be substituted with 1-1/2^16. Indeed, as soon as the returned value of the function is of the type SQ0.15, which has smaller resolution 1/2^15 than the att, the output signal will be constantly 0 when att is specified as 1-1/2^16 as well as it would be specified as 1 exactly.

Assigns the generator frequency.

Parameters
[in,out]pgen– pointer to a generator descriptor object.
[in]freq– a new value of the oscillator frequency in terms of the sampling frequency - i.e., the ratio Fo/Fs, where Fo is the oscillator fequency and Fs is the sampling frequency, both in hertz.

The allowed values of freq are the subset of UQ0.16 values in the discrete range [0.0; 0.25] with resolution of 1/2^16. This range corresponds to the floating point range [0; 0.25] with resolution of 1/2^16:

Fo/Fs fixed point value container code
0.25 0.25 0x4000
0.25 - 1/2^16 0.2499847412109375 0x3FFF
1/2^16 0.0000152587890625 0x0001
0 0.0 (*) 0x0000
Note
If freq equals 0, the generator is paused and returns the same output signal level after each sampling step. The momentary phase is not propagated during this period.

Assigns the generator phase.

Parameters
[in,out]pgen– pointer to a generator descriptor object.
[in]phi– a new value of the oscillator current phase.

The allowed values of phi are the set of UQ0.16 values in the discrete range [0.0; 1.0-1/2^16] with resolution of 1/2^16. This range is scaled to the floating point range [0; 2*pi) with resolution of pi/2^15 radian:

phase, radian fixed point value container code
2*pi not allowed (*) not allowed
2*pi - pi/2^15 0.9999847412109375 0xFFFF
3*pi/2 0.75 0xC000
pi 0.5 0x8000
pi/2 0.25 0x4000
pi/2^15 0.0000152587890625 0x0001
0 0.0 0x0000
Note
Values of the phase outside the domain range [0; 2*pi) radians shall be projected into the domain with the formula phi = phase-2*pi*k, where k is an integer.

§ gen_step()

void gen_step ( struct gen_descr_t *const  pgen)

Propagates the generator state for one sampling step.

Parameters
[in,out]pgen– pointer to a generator descriptor object.

Each time this function is called the current phase of the oscillator is incremented by 2*pi*Fo/Fs radians.

Note
For the generator to work properly this function shall be called exactly one time per each sampling period.
